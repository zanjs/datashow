## Data show

滚屏数据展现动画

## 使用

### 自定义配置参数

```
<div class="block block-2x block-kiwi" data-show="enter bottom over 1s and move 300px after 0.3s">

```


```
defaults: {
    after: "0s",
    enter: "bottom",
    move: "24px",
    over: "0.66s",
    easing: "ease-in-out",
    opacity: 0,
    viewportFactor: .33,
    reset: !1,
    init: !0
}
```
